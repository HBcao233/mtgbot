Installation

  1. clone the repo and install requirement

1git clone https://github.com/HBcao233/mtgbot
2python -m pip install -r requirement.txt
3
4# 安装 ffmpeg (推荐)
5pkg install ffmpeg
6yum install ffmpeg

1# 查看包安装目录 (显示信息中的 Location 项)
2pip show telethon
3cp -r libs/telethon <site-package-location>
