filters 模块

class filters.Filter(value: callable, /)

Bases: object

事件过滤器

可与同类进行:

  • 非运算 ~ Filter

  • 与运算 Filter & Filter

  • 或运算 Filter | Filter

  • 异或运算 Filter ^ Filter

Arguments
value (Callable[[Any], bool]):

过滤函数

参数需为 event, 返回值为 bool

Members
