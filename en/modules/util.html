parse options, find expected string in text split by space

Such as Options(‘/pid mask’, mark=(‘spoiler’, ‘遮罩’), hide=’’) -> Options(mark=True, hide=False)

Arguments
text (str):

text need to parse

args (Mapping[str, Union[str, Sequence[str]]]):

expected strings

In the following circumstances, the attribute Options.aaa = True

  • aaa='': find ‘aaa’

  • aaa='bbb': find ‘aaa’ or ‘bbb’

  • aaa=('bbb', 'ccc'): find ‘aaa’ or ‘bbb’ or ‘ccc’

Arguments

Example

  await asyncio.sleep(1)

    event.chat_id,
    media,
    reply_to=event.message,
    progress_callback=bar.update,
  )
